# TCAX

A Python scripting tool for creating karaoke subtitle effects.

Written in C and Python.

Run on Microsoft Windows (XP/Vista/7/etc.)

Visit TCAX homepage www.tcax.org for more info.

## Dependency

- c++14
- boost c++ libraries
- libpng
- freetype2
- zlib
- python3 or later
